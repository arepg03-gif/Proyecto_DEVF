## Pasteles – Delicias y Tradición

Página web informativa sobre tipos de pasteles, colores, rellenos e historia

## Descripción

Este proyecto es una página web estática desarrollada en **HTML5** y **CSS** que presenta una guía visual y sencilla sobre el mundo de los pasteles. Incluye información sobre tipos comunes, colores más usados en pastelería para fiestas, rellenos populares y una breve reseña histórica.

El diseño utiliza **etiquetas semánticas**, **CSS moderno** y un estilo elegante basado en tarjetas y grillas, con imágenes representativas obtenidas de Unsplash.

----------

## Características principales

-   **Diseño responsive** adaptado para dispositivos móviles.
    
-   **Encabezado visual** con imagen de fondo y título destacado.
    
-   Secciones estructuradas en tarjetas:
    
    -   Tipos de pasteles
        
    -   Colores más usados
        
    -   Rellenos populares
        
    -   Historia de los pasteles
        
-   **Sistema de grillas** para mostrar elementos de forma ordenada.
    
-   **Paleta de colores personalizada** usando variables CSS.
    
-   Imágenes de uso libre (Unsplash).
    

----------

## Tecnologías utilizadas

-   **HTML5**
    
-   **CSS (en línea dentro del archivo HTML)**
    
-   **Responsive Design / Media Queries**
    

----------

## Estructura del archivo

El proyecto consta de un único archivo HTML que contiene:

-   `<header>` con imagen de fondo y título principal.
    
-   `<main>` dividido en cuatro secciones semánticas:
    
    -   `section#tipos`
        
    -   `section#colores`
        
    -   `section#rellenos`
        
    -   `section#historia`
        
-   `<footer>` con créditos y fuente de imágenes.
    
-   Estilos integrados en la etiqueta `<style>`.
